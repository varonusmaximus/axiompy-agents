# axiompy-agents

Distribution **`axiompy-agents`** adds `axiompy.reasoning` and `axiompy.agents` (merged product; import paths unchanged).

Requires **`axiompy`>=2** with server and HTTP-related extras used by RAG and webhooks.

```bash
pip install -e packages/axiompy
pip install -e "packages/axiompy-agents[rag,test-all]"
```

See the [repository README](../../README.md).
